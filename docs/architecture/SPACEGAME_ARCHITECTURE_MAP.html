SpaceGame — архитектурная карта

1. Временная шкала эпох A–K

Главная ось карты: время → эпохи → периоды → системы → переходы. Карточки строятся из data-файла.

2. Закон темпа времени

Историческая шкала задает последовательность и масштаб, а игровой темп определяется плотностью решений.

Быстрый старт

Первые эпохи должны быстро дать ощущение прогресса: от ручного выживания к стоянке, запасу и общему труду.

Нарастающая сложность

Каждый исторический слой добавляет системы: людей, знания, инфраструктуру, энергию, логистику и управление.

Долгий финал

Поздняя игра не обязана иметь жесткий конец: космос и дальние колонии дают длительные проекты и открытое развитие.

3. Матрица систем по эпохам

Системы появляются и становятся центральными в разные эпохи. Матрица строится из массива systems.

4. Слой зависимостей

Первые связи между системами. Этот слой нужен для проектирования условий, блокировок и переходов.

5. Дорожная карта карты

Верхнеуровневые шаги развития самой архитектурной карты.

6. UI-экраны

Начальный индекс экранов, которые должны расти вместе с масштабом игры.

Редактируйте данные в spacegame-architecture-data.js. HTML остается просмотрщиком, CSS — визуальным слоем.
