1. Временная шкала эпох A–K

Главная ось: время → эпохи → периоды → системы → переходы. Данные: spacegame-architecture-data.js.

2. Закон темпа времени

Историческая шкала задаёт последовательность; игровой темп — плотность решений.

Быстрый старт

Первые эпохи дают быстрый прогресс: от ручного выживания к стоянке, запасу и общему труду.

Нарастающая сложность

Каждый слой добавляет системы: людей, знания, инфраструктуру, энергию, логистику и управление.

Долгий финал

Поздняя игра может оставаться открытой: космос и дальние колонии — длительные проекты.

3. Матрица систем по эпохам

Системы появляются и становятся центральными в разные эпохи.

4. Слой зависимостей

Проектные связи между системами: условия, блокировки, переходы.

5. Дорожная карта карты

Развитие самой архитектурной карты как инструмента.

6. UI-экраны

Индекс экранов по масштабу игры.

Редактируйте данные в spacegame-architecture-data.js. HTML — просмотрщик, CSS — визуальный слой.
