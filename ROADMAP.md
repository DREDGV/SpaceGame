# 🚀 Space Economy - Дорожная Карта Проекта

## Общее описание игры
Космическая экономическая стратегия с элементами исследования, торговли, добычи ресурсов, строительства и конструирования. Игрок управляет космической цивилизацией, начиная с одной ракеты и развиваясь до межгалактической империи.

## 🎯 Приоритеты MVP (Minimum Viable Product)

Для первой играбельной версии **минимально необходимо**:

1. ✅ Базовая физика и гравитация
2. ✅ Ракета с топливом
3. ⬜ 5-10 планет с ресурсами
4. ⬜ Система торговли (купить/продать)
5. ⬜ 10+ видов ресурсов
6. ⬜ Конструктор ракет (базовый)
7. ⬜ Система миссий (5+ миссий)
8. ⬜ HUD и главное меню
9. ⬜ Система сохранений
10. ⬜ Звуки и музыка (базовые)

## 🛠 Технический стек

- **Движок:** Pygame 2.6.1
- **Язык:** Python 3.10+
- **Архитектура:** ECS (Entity Component System)
- **Конфигурация:** JSON файлы
- **Данные:** SQLite (сохранения, экономика)
- **Графика:** PNG спрайты + программная генерация
- **Звук:** OGG/MP3
- **Версионирование:** Git

## 📝 Замечания и идеи

### Потенциальные фичы для обсуждения:
- Квантовая телепортация между секторами
- Система погоды на планетах
- Эволюция цивилизации
- Искусственный интеллект помощник
- Голосовое управление
- VR поддержка
- Генерация процедурных секторов

### Балансировка:
- Начальные ресурсы должны позволять первые 2-3 миссии
- Каждая следующая миссия сложнее, но прибыльнее
- Экономика должна быть стабильной, но с возможностями для спекуляции

### Атмосфера:
- Чувство одиночества в космосе
- Величие открытых пространств
- Азарт торговли и исследования
- Удовлетворение от роста империи
