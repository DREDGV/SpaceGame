# SpaceGame — карта эпох (Era Progression Map)

Карта задаёт **смысловые эпохи** от выживания до космоса. Нумерация и точные названия в данных игры могут отличаться; здесь — проектировочный ориентир для одного сквозного прогресса.

Легенда для «будущих систем ещё не здесь»: перечислено то, что **не должно доминировать** в данной эпохе (чтобы не раздувать ранний контент и не смешивать жанр).

### Единая линейка эпох A–G

| Буква | Каноническое название |
|-------|------------------------|
| **A** | Раннее выживание |
| **B** | Стоянка и ранняя община |
| **C** | Поселение и ремесленный город |
| **D** | Государство и территория |
| **E** | Индустрия и энергия |
| **F** | Планетарная цивилизация |
| **G** | Космос |

Мост от текущего прототипа к C: `A_B_TO_C_BRIDGE.md`.

---

## Эпоха A — Раннее выживание

| Поле | Содержание |
|------|----------------|
| **Название** | Раннее выживание |
| **Смысл** | Ограниченные действия, тело, ноша, первые связи материалов |
| **Игровой фокус** | Энергия, срочные нужды, первые находки, пролог как смысл, а не как вечный режим |
| **Ключевые системы** | Персонаж, ручной труд, минимальный крафт, первые «озарения», локальная карта как место, а не как мир |
| **Ресурсы** | Сырьё шагов (ветки, камень, волокно, вода, еда) — мало типов, много смысла в связках |
| **Труд** | Индивидуальный; очередь работ как ритм, не как фабрика |
| **Постройки** | Временные / символические узлы (костёр как якорь стоянки) |
| **Переход дальше** | Закреплённый центр стоянки + устойчивый цикл тепла/еда + первые инструменты и память общины |
| **Ещё не здесь** | Рынки, налоги, армии, многослойная промышленность, глобальная карта |

---

## Эпоха B — Стоянка и ранняя община

| Поле | Содержание |
|------|----------------|
| **Название** | Стоянка и ранняя община |
| **Смысл** | Общий узел, ритм суток, разделение ролей на уровне «кто что делает сегодня» |
| **Игровой фокус** | Постройки-узлы, склады, простые автоматизации, короткие производственные цепочки |
| **Ключевые системы** | Здания с циклами, дневной цикл, логистика «с участка в лагерь», исследования как закрепление практик |
| **Ресурсы** | Полуфабрикаты, топливо, базовые сплавы/керамика (по мере открытий) |
| **Труд** | Несколько «ролей» или приоритетов; население как абстракция или малочисленные «единицы общины» |
| **Постройки** | Мастерская, склад, заготовки под обжиг/печь, простые улучшения |
| **Переход дальше** | Устойчивое производство (не разовые крафты), контролируемые процессы (огонь, обжиг), фундамент знаний |
| **Ещё не здесь** | Городская планировка, фракции-государства, фондовый рынок, орбитальные заводы |

---

## Эпоха C — Поселение и ремесленный город

| Поле | Содержание |
|------|----------------|
| **Название** | Поселение и ремесленный город |
| **Смысл** | Плотность, районы, ремёсла, внутренняя специализация |
| **Игровой фокус** | Расширение карты поселения, цепочки «сырьё → цех → склад → потребление» |
| **Ключевые системы** | Районы или кварталы, санитария/вода как давление, торговля внутри и с соседями на простом уровне |
| **Ресурсы** | Больше промежуточных видов; качество и сортiment |
| **Труд** | Занятость по отраслям; обучение / подготовка кадров в упрощённом виде |
| **Постройки** | Цеха, рынки, стены, мастерские цепочек |
| **Переход дальше** | Стабильный излишек, институты (власть/закон), внешняя угроза или выгода требует государства |
| **Ещё не здесь** | Полноценный планетарный менеджмент, космодромы как основной UI |

---

## Эпоха D — Государство и территория

| Поле | Содержание |
|------|----------------|
| **Название** | Государство и территория |
| **Смысл** | Границы, армия как страховка, налоги и перераспределение, внешняя торговля |
| **Игровой фокус** | Несколько поселений или регионов; логистика между ними; конфликт/дипломатия как расходники |
| **Ключевые системы** | Управление, безопасность, внешняя торговля, дороги/магистрали |
| **Ресурсы** | Стратегические (металл, лошади, деньги/учёт), военные припасы |
| **Труд** | Мобилизация, профессиональные слои, бюрократия как снижение микроменеджмента |
| **Постройки** | Администрация, казармы, дороги, порты (если вода) |
| **Переход дальше** | Индустриальные предпосылки: энергия масштаба, машины, «заводской» ритм |
| **Ещё не здесь** | Межпланетная колонизация как основной геймплей |

---

## Эпоха E — Индустрия и энергия

| Поле | Содержание |
|------|----------------|
| **Название** | Индустрия и энергия |
| **Смысл** | Энергия, масштаб, загрязнение/ресурсные ловушки, транспорт |
| **Игровой фокус** | Энергосистема, фабрики, железные дороги/эквивалент, научно-технический скачок |
| **Ключевые системы** | Энергия, транспорт, производство как сети, исследования как НИОКР |
| **Ресурсы** | Топливо, электричество (или аналог), химия, сплавы |
| **Труд** | Смены, автоматизация линий, профсоюзы/регуляции как *фон* решений |
| **Постройки** | Электростанции, заводы, хабы логистики |
| **Переход дальше** | Планетарные проекты, глобальные цепочки, подготовка к выходу за пределы биосферы |
| **Ещё не здесь** | Фэнтези-магия как замена физики; космос без инженерного моста |

---

## Эпоха F — Планетарная цивилизация

| Поле | Содержание |
|------|----------------|
| **Название** | Планетарная цивилизация |
| **Смысл** | Глобальные ограничения биосферы, климат, добыча, инфраструктура «на всю планету» |
| **Игровой фокус** | Крупные проекты, орбитальная подготовка, глобальная логистика |
| **Ключевые системы** | Планетарная карта/регионы, экология как давление, космоподготовка |
| **Ресурсы** | Редкие элементы, орбитальные материалы, высокие энергозатраты |
| **Труд** | Высокая автоматизация; игрок — макро |
| **Постройки** | Космодромы, ускорители, орбитальные верфи (подготовительные) |
| **Переход дальше** | Устойчивая орбита, добыча в космосе, межпланетные цепочки |
| **Ещё не здесь** | Возврат к «одному герою собирает ветки» как основной петле |

---

## Эпоха G — Космос

| Поле | Содержание |
|------|----------------|
| **Название** | Космос |
| **Смысл** | Орбита и дальше как **продолжение экономики**: энергия, добыча, транспорт, риск |
| **Игровой фокус** | Орбитальные цепочки, запуски, обслуживание, дальняя логистика |
| **Ключевые системы** | Космос (как система в матрице), энергия, транспорт, исследования, управление |
| **Ресурсы** | Реагенты орбиты, компоненты двигателей, радиационная/ремонтная инфраструктура |
| **Труд** | Экипажи, дроны, дистанционное управление; население — через специализацию |
| **Постройки** | Станции, верфи, площадки добычи |
| **Переход дальше** | Контентные расширения (другие тела, фракции) — **после** закрепления костяка |
| **Ещё не здесь** | Бесконечный дроп без новых цепочек; «новая игра» без экономической связи с прошлым |

## Связь с текущим прототипом

Сейчас в данных явно заданы **`primitive`** и заготовка **`early_production`** (`world.js`). Экономические стадии Stage 1A (`lone_survivor` → `campfire_camp` → `early_workshop`) покрывают только **начало эпохи A–B**. Этот документ задаёт, **куда** эти куски должны стыковаться дальше — без требования немедленной реализации. Проектировочный мост к эпохе **C**: `A_B_TO_C_BRIDGE.md`.
