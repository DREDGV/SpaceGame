# Building Art

Put camp building images in this folder.

Expected filenames for 5-level building art:

- `campfire_lvl1.png` ... `campfire_lvl5.png`
- `shelter_lvl1.png` ... `shelter_lvl5.png`
- `storage_lvl1.png` ... `storage_lvl5.png`
- `workshop_lvl1.png` ... `workshop_lvl5.png`
- `kiln_lvl1.png` ... `kiln_lvl5.png`

Recommended format: transparent PNG or WebP, square canvas, 256x256 or 512x512.
The camp UI falls back to `web/prototype/assets/icons/` while level art is missing here.
